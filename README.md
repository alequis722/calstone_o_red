# Calstone O Red
## Introduction
Calstone O Red is a lisp-like toy programming language written in rust.

## Built-in Functions
| Name | No. of arguments |
|:---|---|
| `print` | >=0 |
| `let` | 2 or 3 |
| `add` | >=2 |
| `sub` | dto |
| `mul` | dto |
| `div` | dto |
| `ln` | 1 |
| `log` | 1 |
| `log_` | 2 |
| `zero` | 1 |
| `pos` | 1 |
| `neg` | 1 |
| `if` | 2 or 3 |
