# Calstone O Red
## Introduction
Calstone O Red is a lisp-like toy programming language written in rust.

## Built-in Functions
| Name | Usage |
|:---:|:---|
| `print` | Prints to terminal |
| `let` | Defines a variable or a function |
| `add` | Adds numbers |
| `sub` | Subtracts numbers |
| `mul` | Multiplies numbers |
| `div` | Divides numbers |
| `log` | Base-10 logarithm |
| `ln` | Natural logarithm |
| `log_` | Base-r logarithm |
